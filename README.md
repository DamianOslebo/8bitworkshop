### 🏠 [Homepage](http://8bitworkshop.com/)

## Install

To build the 8bitworkshop IDE:

```sh
git submodule init
git submodule update
npm i
npm run build
```

## Usage

Start a web server on http://localhost:8000/ while TypeScript compiles in the background:

```sh
make tsweb
```

## Run Tests

```sh
npm test
```
